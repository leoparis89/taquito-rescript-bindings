# `Taqito Rescript Bindings`

## Status

⚠️ Work in progress

## Installation

```console
yarn add taquito-rescript-bindings
```

`taquito-rescript-bindings` should be added to `bs-dependencies` in your
`bsconfig.json`:

```diff
{
  //...
  "bs-dependencies": [
    // ...
+    "taquito-rescript-bindings"
  ],
  //...
}
```
